## 📝 开发建议

### 添加新页面
1. 在 `src/views` 对应目录下创建 `.vue` 文件
2. 在 `src/router/index.js` 中添加路由配置
3. 如需 API 调用，在 `src/api` 中创建对应接口文件

### 调用 API 示例
```vue
<script setup>
import { ref, onMounted } from 'vue'
import { getAdminList } from '@/api'

const tableData = ref([])

onMounted(async () => {
  const res = await getAdminList({ page: 1, pageSize: 10 })
  tableData.value = res.data.list
})
</script>
```

### 使用状态管理
```vue
<script setup>
import { useUserStore } from '@/store/user'

const userStore = useUserStore()

// 获取用户信息
console.log(userStore.userInfo)

// 调用登录
await userStore.login('root', '123456')

// 退出登录
userStore.logout()
</script>
```

## 🐛 常见问题

**Q: 登录后看不到用户信息？**
A: 确保后端服务正常运行，并且返回正确的用户信息格式

**Q: 页面刷新后需要重新登录？**
A: Token 已存储在 localStorage，应该会自动保持登录状态。检查 `mall_token` 是否存在

**Q: API 请求失败？**
A: 检查 `.env.development` 中的 API 地址是否正确，确保后端服务已启动
